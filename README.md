# Cinderhaven Data Platform

Modern data platform for Cinderhaven Provisions, a fictional $25M specialty food brand. Demonstrates source-to-mart pipelines, data quality testing, orchestration, and lineage for CPG data shapes — product master, retailer transactions, deductions, POS, and EDI.

This is the warehouse substrate that powers every downstream analytical portfolio piece.
